Jenkins First Job


    Step 1: Open Jenkins Dashboard
    Dashboard → Click "New Item"

    Step 2: Create a New Job
    Enter Job Name: first-job
    Select: Freestyle Project
    Click: OK

    Step 3: Configure Job
    Description: This is my first Jenkins job.

    Step 4: Build Trigger (Optional)
    Build Triggers → Poll SCM 
    (This step is optional for the first test build.)

    Step 5: Add Build Step
    Scroll to: Build → Add Build Step → Execute Shell

    Step 6: Add Shell Commands
    echo "Hello Jenkins"
    date
    whoami

    Step 7: Save Job
    Click "Save"

    Step 8: Run the Job
    Click "Build Now"

    Step 9: Check Output
    Open Build History → Click #1 → Console Output

    Expected Output:
    Hello Jenkins
    Fri Mar 6
    jenkins
    Finished: SUCCESS

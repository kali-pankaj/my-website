1. Self-Awareness

  • Pause करना सीखो (5–10 सेकंड रुक जाओ)
  • Daily self-check:
    • आज मैंने कहाँ अच्छा behave किया?
    • कहाँ मैं गलत था?
  • Trigger पहचानो (गुस्सा कब आता है)
  • Feedback लो
  • Journal लिखो (आज क्या हुआ)
  • Emotion पहचानो: irritation, jealousy, insecurity

2. बोलने से पहले सोचो

  • क्या ये जरूरी है?
  • क्या मैं सही तरीके से बोल रहा हूँ?
  • क्या इससे सामने वाले को hurt होगा?
  • क्या मैं calm हूँ?

3. Ego Control

  • हर बार खुद को सही मत मानो
  • Sorry बोलना सीखो
  • Listen more
  • Compare मत करो
  • Feedback accept करो
  • Argument जीतना जरूरी नहीं
  • Gratitude practice करो

4. Respectful Language

“मुझे लगता है हम ऐसे try कर सकते हैं”

बात करने का तरीका बदलो, attitude खुद बदल जाएगा

5. गुस्सा Control

  • Pause Rule
  • Deep Breathing (4-4-6)
  • Situation से दूर जाओ
  • Reframe सोच
  • Calm होकर बोलो
  • Triggers पहचानो
  • Daily Practice

6. Empathy

  • Active Listening
  • खुद को उसकी जगह रखो
  • सवाल पूछो
  • Judge मत करो
  • Feelings acknowledge करो

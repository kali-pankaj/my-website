Introduction to Python

Python is a powerful, simple, and beginner-friendly programming language. It is widely used in DevOps, Automation, Web Development, Data Science, Artificial Intelligence, and Cloud Computing.

What is Python?

Python is a high-level, interpreted programming language created by Guido van Rossum in 1991. It focuses on readability and simplicity, making it perfect for beginners and professionals.

Strings

A string is a sequence of characters used to store and manipulate text in Python. Strings are enclosed in single quotes (' '), double quotes (" "), or triple quotes (""" """).

String Indexing

Indexing means accessing individual characters of a string using their position. Python strings are zero-indexed, meaning the first character is at index 0.

  • Positive Indexing: Starts from 0 (left to right)
  • Negative Indexing: Starts from -1 (right to left)
text = "PYTHON"

# Positive indexing
print(text[0])   # P
print(text[1])   # Y

# Negative indexing
print(text[-1])  # N
print(text[-2])  # O

String Slicing

Slicing is used to extract a part (substring) of a string. It follows the syntax: string[start:end:step]

  • start: Starting index (included)
  • end: Ending index (not included)
  • step: Jump size (optional)
text = "PYTHON PROGRAM"

print(text[0:6])     # PYTHON
print(text[7:14])    # PROGRAM
print(text[:6])      # PYTHON
print(text[::2])     # PTOPRGM (every 2nd character)

String Methods

Python provides built-in methods to modify and analyze strings.

  • upper() → Converts string to uppercase
  • lower() → Converts string to lowercase
  • strip() → Removes whitespace from both ends
  • replace(old, new) → Replaces characters or words
  • find() → Returns index of first occurrence
  • len() → Returns length of string
text = "  Hello Python  "

print(text.upper())       # HELLO PYTHON
print(text.lower())       # hello python
print(text.strip())       # Hello Python
print(text.replace("Python", "World"))

print(len(text))          # length of string
print(text.find("Python"))

Formatting (f-strings)

f-strings allow embedding variables directly inside strings for easy formatting. They were introduced in Python 3.6.

  • Prefix string with f
  • Use curly braces {} for variables
  • Supports expressions inside braces
name = "Kali"
age = 20

print(f"My name is {name}")
print(f"I am {age} years old")

# Expression inside f-string
print(f"Next year age will be {age + 1}")

Real-Life Example

student = "Aman"
marks = 85

print(f"Student {student} scored {marks} marks in Python exam.")

Input and Output

Input and output are fundamental parts of any programming language. They allow interaction between the user and the program. In Python, we use input() for taking input and print() for displaying output.

What is Input?

Input means receiving data from the user during program execution. Python uses the input() function to take input from the keyboard.

  • Takes data from user during runtime
  • Always returns data as a string
  • Can include a message (prompt) inside brackets

Input Syntax

variable = input("Prompt message")

Input Example

name = input("Enter your name: ")
age = input("Enter your age: ")

print(name)
print(age)

Important Note About Input

  • input() always returns a string (even numbers)
  • To use numbers, convert using int() or float()
age = input("Enter age: ")
print(type(age))   # str

age = int(input("Enter age: "))
print(type(age))   # int

What is Output?

Output means displaying data to the user. Python uses the print() function to show output on the screen.

  • Displays results of program execution
  • Can print text, variables, or expressions
  • Supports multiple values in one line

Print Syntax

print(value1, value2, ...)

Print Example

print("Hello World")
print("Python is easy to learn")

name = "Kali"
print("My name is", name)

Example Program (Input + Output)

This program takes user input and displays a greeting message.

Advanced Example

You can also combine input with numbers and calculations. 

num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))

sum = num1 + num2

print("Sum is:", sum)

Common Mistakes

Conditional Statements

Conditional statements are used to make decisions in a program. They execute different blocks of code based on conditions (True or False). Python provides if, elif, else, and nested if statements.

if Statement

The if statement executes a block of code only when the condition is True.

  • Used for single condition checking
  • Executes only if condition is True
age = 18

if age >= 18:
    print("You are eligible to vote")

if-else Statement

The else block executes when the if condition is False.

  • Used when two outcomes are possible
  • Either if or else will execute
age = 16

if age >= 18:
    print("You can vote")
else:
    print("You cannot vote")

if-elif-else Statement

The elif (else if) is used when multiple conditions need to be checked.

  • Used for multiple conditions
  • Checks conditions one by one
  • Only one block will execute
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")

Nested if Statement

A nested if is an if statement inside another if statement. It is used when multiple conditions depend on each other.

  • One if inside another if
  • Used for complex decision making
age = 20
citizen = True

if age >= 18:
    if citizen:
        print("You are eligible to vote")
    else:
        print("You must be a citizen to vote")
else:
    print("You are not eligible to vote")

Important Notes

  • Conditions must return True or False
  • Indentation is very important in Python
  • Only one block executes in if-elif-else chain

Loops

Loops are used to execute a block of code multiple times. They help avoid repetition and make programs efficient. Python provides for loop and while loop, along with control statements like break, continue, and pass.

for Loop

The for loop is used to iterate over a sequence like list, string, or range.

  • Used when number of iterations is known
  • Works with sequences (list, string, range)
for i in range(5):
    print(i)

Example with list: 

fruits = ["apple", "banana", "mango"]

for fruit in fruits:
    print(fruit)

while Loop

The while loop runs as long as a condition is True.

  • Used when number of iterations is unknown
  • Runs until condition becomes False
i = 0

while i < 5:
    print(i)
    i += 1

break Statement

The break statement is used to stop the loop immediately.

  • Terminates loop execution
  • Used when condition is met early
for i in range(10):
    if i == 5:
        break
    print(i)

continue Statement

The continue statement skips the current iteration and moves to the next one.

  • Skips current loop iteration
  • Does not stop the loop
for i in range(5):
    if i == 2:
        continue
    print(i)

pass Statement

The pass statement is a placeholder that does nothing. It is used when code is required syntactically but no action is needed.

  • Used as a placeholder
  • Does nothing when executed
for i in range(5):
    if i == 2:
        pass
    print(i)

Important Notes

  • for loop is used for known iterations
  • while loop depends on condition
  • break stops loop completely
  • continue skips current iteration
  • pass does nothing (placeholder)

Data Structures

Data structures are used to store and organize data in Python. Python provides built-in data structures like Lists, Tuples, Sets, and Dictionaries.

Lists

A list is an ordered, changeable collection that allows duplicate values.

  • Ordered (index-based)
  • Mutable (can be changed)
  • Allows duplicates
fruits = ["apple", "banana", "mango"]
print(fruits)

List Methods

  • append() → Adds item at end
  • insert() → Adds item at specific position
  • remove() → Removes specific item
  • pop() → Removes item by index
fruits = ["apple", "banana", "mango"]

fruits.append("orange")
fruits.insert(1, "grapes")
fruits.remove("banana")
fruits.pop(0)

print(fruits)

Tuples

A tuple is an ordered collection that is immutable (cannot be changed after creation).

  • Ordered
  • Immutable (unchangeable)
  • Allows duplicates
colors = ("red", "green", "blue")
print(colors)

Tuples are used when data should not be modified.

Sets

A set is an unordered collection of unique elements.

  • Unordered
  • No duplicate values allowed
  • Mutable (can add/remove items)
numbers = {1, 2, 3, 3, 4}
print(numbers)   # duplicates removed

Set Methods

  • add() → Adds element
  • remove() → Removes element
numbers = {1, 2, 3}

numbers.add(4)
numbers.remove(2)

print(numbers)

Dictionaries

A dictionary stores data in key-value pairs.

  • Key-value structure
  • Keys must be unique
  • Values can be any type
student = {
    "name": "Kali",
    "age": 20,
    "course": "Python"
}

print(student)
print(student["name"])

Dictionary Methods

  • keys() → Returns all keys
  • values() → Returns all values
  • items() → Returns key-value pairs
print(student.keys())
print(student.values())
print(student.items())

Important Notes

  • Lists are mutable and ordered
  • Tuples are immutable
  • Sets store unique values only
  • Dictionaries use key-value mapping

Functions

Functions are reusable blocks of code that perform a specific task. They help in making programs modular, readable, and reusable.

Defining Functions

In Python, a function is defined using the def keyword.

  • Starts with def keyword
  • Function name should be meaningful
  • Code inside function must be indented
def greet():
    print("Hello, Welcome to Python!")

greet()

Functions with Arguments

Arguments are values passed to a function when it is called. They allow functions to work with different inputs.

  • Arguments are passed inside parentheses
  • Can have multiple arguments
def greet(name):
    print("Hello", name)

greet("Kali")
greet("Aman")

Return Values

The return statement is used to send a result back from a function.

  • Returns output from function
  • Ends function execution
def add(a, b):
    return a + b

result = add(5, 3)
print("Sum is:", result)

Types of Arguments

  • Positional arguments: Passed in order
  • Keyword arguments: Passed with parameter name
def student(name, age):
    print(name, age)

student("Kali", 20)          # positional
student(age=21, name="Aman") # keyword

Lambda Functions

Lambda functions are small anonymous functions defined in a single line.

  • Used for simple operations
  • No function name required
  • Defined using lambda keyword
square = lambda x: x * x

print(square(5))

Lambda with Multiple Arguments

add = lambda a, b: a + b

print(add(10, 5))

Important Notes

  • Functions make code reusable
  • Arguments pass input to functions
  • Return gives output from function
  • Lambda is used for short functions

Modules and Packages

Modules and packages are used to organize Python code into reusable files. They help break large programs into smaller, manageable parts.

What is a Module?

A module is a Python file that contains functions, variables, or classes that can be reused in other programs.

  • Each Python file (.py) is a module
  • Helps in code reusability
  • Reduces code duplication

Importing a Module

The import keyword is used to include a module in your program. 

import math

print(math.sqrt(16))

You can also import specific functions: 

from math import sqrt

print(sqrt(25))

Built-in Modules

Python provides many built-in modules for common tasks.

math Module

  • Used for mathematical operations
  • Provides functions like sqrt, pow, ceil
import math

print(math.sqrt(49))
print(math.pow(2, 3))
print(math.ceil(4.3))

random Module

  • Used to generate random numbers
  • Useful for games and simulations
import random

print(random.randint(1, 10))
print(random.random())

os Module

  • Used to interact with the operating system
  • Helps manage files and directories
import os

print(os.getcwd())  # current directory

Creating Your Own Module

You can create your own module by saving Python code in a .py file and importing it in another file.

  • Create a file with .py extension
  • Define functions or variables inside it
  • Import it into another program

Example: mymodule.py

# mymodule.py

def greet(name):
    print("Hello", name)

def add(a, b):
    return a + b

Using Custom Module

import mymodule

mymodule.greet("Kali")

print(mymodule.add(5, 3))

Important Notes

  • Modules help organize code
  • Built-in modules save time
  • Custom modules improve reusability
  • Packages are collections of modules

File Handling

File handling is used to store data permanently in files. Python allows us to create, read, write, and append files easily using built-in functions.

Opening a File

The open() function is used to open a file in Python.

  • Syntax: open(filename, mode)
  • Mode defines the purpose (read, write, append)
f = open("file.txt", "r")

File Modes

  • r → Read mode (default)
  • w → Write mode (creates new file or overwrites existing)
  • a → Append mode (adds data at end of file)
  • r+ → Read and write mode

Reading a File

The read() method is used to read the content of a file. 

f = open("file.txt", "r")
print(f.read())
f.close()

You can also read line by line: 

f = open("file.txt", "r")

print(f.readline())
print(f.readline())

f.close()

Writing to a File

The write() method is used to write data into a file. It overwrites existing content. 

f = open("file.txt", "w")
f.write("Hello Python!")
f.close()

Appending to a File

The append mode (a) is used to add data at the end of the file without deleting existing content. 

f = open("file.txt", "a")
f.write("\nNew line added")
f.close()

Best Practice: Using with Statement

The with statement automatically closes the file after use. It is the recommended way to handle files. 

with open("file.txt", "r") as f:
    print(f.read())

Important Notes

  • Always close files after opening
  • Use "w" carefully (it deletes old data)
  • "a" is safe for adding data
  • with statement is best practice

Error Handling

Error handling is used to handle runtime errors in Python without stopping the program. It helps make programs more stable and user-friendly.

What is an Error?

An error occurs when a program cannot execute properly due to invalid input, logic mistakes, or system issues.

  • Errors stop program execution
  • Common errors: division by zero, invalid input, file not found

try and except Block

The try block contains code that might cause an error. The except block handles the error if it occurs.

  • Prevents program crash
  • Handles errors gracefully
try:
    num = int(input("Enter number: "))
    print(10 / num)

except:
    print("Something went wrong!")

Handling Specific Errors

You can handle specific types of errors for better control. 

try:
    num = int(input("Enter number: "))
    print(10 / num)

except ZeroDivisionError:
    print("Cannot divide by zero")

except ValueError:
    print("Invalid input! Please enter a number")

finally Block

The finally block always executes, whether an error occurs or not. It is commonly used for cleanup tasks like closing files.

  • Always executes
  • Used for cleanup operations
try:
    num = int(input("Enter number: "))
    print(10 / num)

except ZeroDivisionError:
    print("Error: Division by zero")

finally:
    print("Execution completed")

Raising Errors

The raise keyword is used to manually generate an error in Python.

  • Used to create custom errors
  • Helps enforce rules in programs
age = int(input("Enter age: "))

if age < 18:
    raise ValueError("Age must be 18 or above")

print("Access granted")

Custom Error Message Example

try:
    password = input("Enter password: ")

    if len(password) < 6:
        raise Exception("Password too short")

except Exception as e:
    print("Error:", e)

Important Notes

  • try block runs risky code
  • except handles errors
  • finally always runs
  • raise creates custom errors

Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a programming style based on objects and classes. It helps organize code into reusable and structured components.

Class and Object

A class is a blueprint, and an object is an instance of a class.

  • Class defines properties and behavior
  • Object is a real-world representation
class Student:
    def show(self):
        print("Hello, I am a student")

s1 = Student()
s1.show()

Constructor (__init__)

The constructor is a special method that runs automatically when an object is created.

  • Used to initialize object values
  • Called automatically
  • Defined using __init__ method
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def show(self):
        print(self.name, self.age)

s1 = Student("Kali", 20)
s1.show()

Inheritance

Inheritance allows a class to acquire properties and methods from another class.

  • Promotes code reuse
  • Parent → Child relationship
class Parent:
    def show(self):
        print("This is parent class")

class Child(Parent):
    def display(self):
        print("This is child class")

c1 = Child()
c1.show()
c1.display()

Polymorphism

Polymorphism means "many forms". It allows the same function name to behave differently.

  • Same function, different behavior
  • Improves flexibility
class Bird:
    def sound(self):
        print("Bird sound")

class Dog:
    def sound(self):
        print("Dog barks")

def make_sound(animal):
    animal.sound()

make_sound(Bird())
make_sound(Dog())

Encapsulation

Encapsulation is the concept of hiding internal data and restricting direct access.

  • Protects data from outside access
  • Uses private variables
  • Improves security
class Bank:
    def __init__(self, balance):
        self.__balance = balance

    def show_balance(self):
        print(self.__balance)

b1 = Bank(1000)
b1.show_balance()

Important Notes

  • Class is a blueprint, object is instance
  • Constructor initializes data
  • Inheritance reuses code
  • Polymorphism allows multiple behaviors
  • Encapsulation protects data

Advanced Python

Advanced Python concepts help you write more efficient, clean, and powerful code. This includes Decorators, Generators, Iterators, and Comprehensions.

Decorators

Decorators are functions that modify the behavior of another function without changing its code.

  • Used to extend functionality
  • Applied using @ symbol
  • Useful in logging, authentication, etc.
def decorator(func):
    def wrapper():
        print("Before function")
        func()
        print("After function")
    return wrapper

@decorator
def hello():
    print("Hello World")

hello()

Generators (yield)

Generators are special functions that return values one at a time using the yield keyword.

  • Efficient for large data
  • Do not store entire data in memory
  • Use yield instead of return
def my_generator():
    yield 1
    yield 2
    yield 3

for value in my_generator():
    print(value)

Iterators

An iterator is an object that allows traversing through all elements one by one.

  • Uses __iter__() and __next__() methods
  • Returns one element at a time
numbers = [1, 2, 3]

it = iter(numbers)

print(next(it))
print(next(it))
print(next(it))

Comprehensions

Comprehensions provide a shorter way to create lists, sets, or dictionaries.

  • More readable and compact
  • Faster than traditional loops

List Comprehension

numbers = [x for x in range(5)]
print(numbers)

With Condition

even = [x for x in range(10) if x % 2 == 0]
print(even)

Dictionary Comprehension

squares = {x: x*x for x in range(5)}
print(squares)

Important Notes

  • Decorators modify functions
  • Generators use yield for memory efficiency
  • Iterators provide step-by-step access
  • Comprehensions make code shorter and faster

Database (Basic)

A database is used to store data permanently in an organized way. Python provides built-in support for SQLite to work with databases easily.

What is SQLite?

SQLite is a lightweight, file-based database system used for small to medium applications. It does not require a separate server.

  • Built into Python (no installation needed)
  • Stores data in a single file
  • Easy to use for beginners

Connecting to SQLite

The sqlite3 module is used to connect Python with SQLite database. 

import sqlite3

conn = sqlite3.connect("database.db")
print("Database connected successfully")

Creating a Table

SQL commands are used to create tables inside the database. 

import sqlite3

conn = sqlite3.connect("database.db")
cursor = conn.cursor()

cursor.execute("""
CREATE TABLE students (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER
)
""")

conn.commit()
conn.close()

CRUD Operations

CRUD stands for Create, Read, Update, and Delete — the basic operations in a database.

1. Create (Insert Data)

import sqlite3

conn = sqlite3.connect("database.db")
cursor = conn.cursor()

cursor.execute("INSERT INTO students (name, age) VALUES ('Kali', 20)")

conn.commit()
conn.close()

2. Read (Fetch Data)

import sqlite3

conn = sqlite3.connect("database.db")
cursor = conn.cursor()

cursor.execute("SELECT * FROM students")

data = cursor.fetchall()

for row in data:
    print(row)

conn.close()

3. Update Data

import sqlite3

conn = sqlite3.connect("database.db")
cursor = conn.cursor()

cursor.execute("UPDATE students SET age = 21 WHERE name = 'Kali'")

conn.commit()
conn.close()

4. Delete Data

import sqlite3

conn = sqlite3.connect("database.db")
cursor = conn.cursor()

cursor.execute("DELETE FROM students WHERE name = 'Kali'")

conn.commit()
conn.close()

Important Notes

  • SQLite is lightweight and beginner-friendly
  • Always commit changes after insert/update/delete
  • Close connection after operations
  • CRUD is the base of all database systems

Web Basics

Python is widely used for web development. It provides powerful frameworks like Flask and Django to build web applications easily.

Flask (Simple Web Framework)

Flask is a lightweight web framework used to build simple and small web applications quickly.

  • Minimal and easy to use
  • Good for beginners
  • Flexible and lightweight

  • Follows MVC/MVT architecture
  • Built-in admin panel
  • Secure and scalable

Django Installation

pip install django

Simple Django View

  • Flask gives flexibility, Django gives structure

Automation / DevOps with Python

Python is widely used in DevOps and automation to manage systems, run commands, and interact with APIs. It helps automate repetitive tasks and improve productivity.

OS Module

The os module is used to interact with the operating system. It helps manage files, directories, and system-level operations.

  • Work with files and folders
  • Get system information
  • Run OS-level operations
import os

print(os.getcwd())   # current directory
os.mkdir("test_folder")
os.listdir()

Common OS Functions

import os

os.rename("old.txt", "new.txt")
os.remove("file.txt")

subprocess Module

The subprocess module allows Python to run system commands and external programs.

  • Execute shell commands
  • Interact with system processes
  • Useful in DevOps automation
import subprocess

subprocess.run(["echo", "Hello World"])

Running System Commands

import subprocess

subprocess.run(["ls"])       # Linux/Mac
subprocess.run(["dir"], shell=True)  # Windows

Automation Scripts

Automation scripts are used to perform repetitive tasks automatically without manual effort.

  • File management automation
  • System cleanup scripts
  • Backup automation
import os

folder = "test_folder"

if not os.path.exists(folder):
    os.mkdir(folder)
    print("Folder created automatically")

Example: File Organizer Script

import os

for file in os.listdir():
    if file.endswith(".txt"):
        print("Text file found:", file)

API Requests (requests library)

The requests library is used to interact with APIs and fetch data from the internet.

  • Used for API communication
  • Fetch data from websites
  • Send HTTP requests (GET, POST)

Installation

pip install requests

GET Request Example

import requests

response = requests.get("https://api.github.com")

print(response.status_code)
print(response.text)

POST Request Example

import requests

data = {"name": "Kali", "age": 20}

response = requests.post("https://httpbin.org/post", data=data)

print(response.json())

API & JSON

APIs (Application Programming Interfaces) allow different applications to communicate with each other. JSON (JavaScript Object Notation) is the most common format used to exchange data between systems.

What is an API?

An API is a bridge that allows one software application to talk to another and exchange data.

  • Used to fetch data from servers
  • Connects frontend and backend systems
  • Returns data in JSON format

Working with APIs (GET Request)

Python uses the requests library to interact with APIs. A GET request is used to fetch data from a server. 

import requests

response = requests.get("https://api.github.com/users")

print(response.status_code)
print(response.text)

API Response Example

{
  "login": "octocat",
  "id": 1,
  "node_id": "MDQ6VXNlcjE"
}

What is JSON?

JSON is a lightweight data format used to store and exchange data between systems.

  • Human-readable format
  • Used in APIs and web services
  • Works like Python dictionaries

JSON Structure

{
    "name": "Kali",
    "age": 20,
    "course": "Python"
}

JSON Parsing in Python

JSON data can be converted into Python dictionaries using the json module. 

import json

data = '{"name": "Kali", "age": 20}'

parsed_data = json.loads(data)

print(parsed_data)
print(parsed_data["name"])

Converting Python to JSON

Python objects can be converted into JSON format using json.dumps(). 

import json

data = {
    "name": "Aman",
    "age": 22
}

json_data = json.dumps(data)

print(json_data)

API + JSON Example

import requests
import json

response = requests.get("https://api.github.com/users")

data = response.json()

for user in data:
    print(user["login"])

Projects (Very Important 🚀)

Projects are the best way to apply Python knowledge in real-world scenarios. They help you improve problem-solving skills and build your portfolio for jobs or internships.

1. Calculator Project

A calculator performs basic arithmetic operations like addition, subtraction, multiplication, and division.

  • Uses functions for operations
  • Takes user input
  • Performs calculations dynamically
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))

print("Sum:", num1 + num2)
print("Difference:", num1 - num2)
print("Product:", num1 * num2)
print("Division:", num1 / num2)

2. To-Do App

A To-Do app helps users manage tasks by adding, viewing, and removing tasks.

  • Uses lists for task storage
  • Supports add and remove operations
  • Simple task management system
tasks = []

tasks.append("Learn Python")
tasks.append("Build Project")

print("Tasks:", tasks)

tasks.remove("Learn Python")
print("Updated Tasks:", tasks)

3. Login System

A login system checks username and password for authentication.

  • Uses conditional statements
  • Validates user credentials
  • Basic security logic
username = input("Enter username: ")
password = input("Enter password: ")

if username == "admin" and password == "1234":
    print("Login successful")
else:
    print("Invalid credentials")

4. File Organizer

A file organizer automatically sorts files into folders based on type.

  • Uses OS module
  • Sorts files by extension
  • Automates file management
import os

for file in os.listdir():
    if file.endswith(".txt"):
        print("Text file:", file)
    elif file.endswith(".jpg"):
        print("Image file:", file)

5. Web Scraper

A web scraper extracts data from websites automatically.

  • Uses requests library
  • Fetches HTML content
  • Extracts useful data
import requests

url = "https://example.com"
response = requests.get(url)

print(response.text[:200])  # first 200 characters

🚀 Kubernetes (kubectl) Cheat Sheet

Cluster & Context

kubectl cluster-info → cluster details
kubectl version --short → version
kubectl config view → config
kubectl config current-context → active cluster
kubectl config use-context minikube → switch cluster

Nodes

kubectl get nodes → list nodes
kubectl describe node minikube → node info

Pods

kubectl get pods → list pods
kubectl logs nginx → logs
kubectl exec -it nginx -- bash → enter pod

Deployments

kubectl create deployment nginx --image=nginx → create app
kubectl scale deployment nginx --replicas=3 → scale

Services

kubectl expose deployment nginx --type=NodePort --port=80 → expose

Namespaces

kubectl create ns dev → create namespace

Config

kubectl create configmap my-config --from-literal=key=value
kubectl create secret generic my-secret --from-literal=password=1234

YAML

kubectl apply -f file.yaml → apply
kubectl delete -f file.yaml → delete

Debug

kubectl describe pod nginx
kubectl get events

Monitoring

kubectl top pods

Labels

kubectl label pod nginx env=dev

Port Forward

kubectl port-forward pod/nginx 8080:80

Copy

kubectl cp nginx:/app/file.txt ./file.txt

All Resources

kubectl get all

Delete

kubectl delete all --all

Short

kubectl get po
kubectl get svc

Watch

kubectl get pods -w

Explain

kubectl explain pod
